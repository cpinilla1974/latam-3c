# LATAM-3C v0

## Iniciar Aplicación

```bash
# Activar entorno virtual
source venv/bin/activate

# Ejecutar aplicación
streamlit run app.py

# Abrir navegador en: http://localhost:8501
```

## Aplicaciones Disponibles

- **Calculadora 3C País**: `streamlit run calculadora-3c-pais/app.py`
- **Formularios Empresas**: `streamlit run formularios-empresas/app.py`
