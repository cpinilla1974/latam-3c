# Opciones de Arquitectura - Sistema Huella de Carbono

## Opción Excel + Web (Híbrido simplificado)
- **Entrada**: Archivos Excel estandarizados por empresa
- **Procesamiento**: Web app que recibe, valida y consolida
- **Ventaja**: Familiaridad, bajo costo entrada
- **Referencia**: Protocolo GCCA

## Descartados
- **Desktop + Web**: Demasiada complejidad, dos proyectos
- **Access**: Problemas de licencias, no todas las empresas lo tienen

## Consideraciones clave
- Minimizar riesgo de proyecto
- Aprovechar herramientas existentes (Excel)
- Centralizar complejidad en un solo lugar (web)
